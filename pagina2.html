HTML5

HTML5 es el sucesor del HTML4.01 y es el último lenguaje de marcado de hipertexto para los sitios
web desarrollados por World Wide Web Consortium (W3C) y Grupo Web de hipertexto Aplicación de
Tecnología de trabajo (WHATWG).

A pesar de que el proyecto de trabajo se publicó en 2008, hasta mayo de 2011, el grupo de trabajo
recién consolidó el HTML5 para su real momento.

Hoy en día, todos los navegadores soportan características de HTML5 y su tecnología y por lo tanto,
los desarrolladores web y diseñadores pueden usar esta tecnología HTML hoy en su mejor momento. En el
siguiente mensaje se describen algunas de las nuevas características de HTML5, elementos de formulario,
almacenamiento local de HTML5 e Internet Explorer.

Nuevos Elementos de formulario:

"datalist": Especifica una lista de opciones predefinidas para los controles de entrada

"keygen": Define un campo generador de claves (passwords) (para formularios)

"output": Define el resultado de un cálculo
